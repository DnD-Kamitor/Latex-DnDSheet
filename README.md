# Latex-DnDSheet
Creating Latex overleaf books. 

- Using the https://github.com/rpgtex/DND-5e-LaTeX-Template.git

objective: 
- ability to make latex characters sheets with DnD Latex format (also offline) with linux. 
- terminal application for: 
    - Character sheet creation DnD rules 2024
    - Character sheet creation dnd rules 5.0 (2014)
    - Character sheet with Traditional Ledger sheets (one for items, tracking gold, spells)
    - Character sheet with extended notes
    - NPC creator 
    - DnD Session Creation  (for the Dungeon Master)
